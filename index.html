Programming Parallel Computers

Engineered algorithm for modern parallel computers. Designed and developed programs that make the best possible use of the computing power of multicore CPUs and GPUs. Learned and implemented Instruction-level parallelism, Vector instructions and memory access pattern optimization. Used libraries such as openMP and CUDA.

1. Implemented a program that is used to find pairwise correlations between pixel rows in PNG images. Developed a CPU baseline model with a runtime of ~9secs and improved it to ~0.2 secs in the final model by using all resources in the GPU.

2. Implemented a program that segmented image in PNG files. Found the best way to partition a given figure in two parts - a monochromatic rectangle and a monochromatic background. The objective was to minimize the sum of squared errors. Baseline CPU model ran in ~3 secs and the improved GPU version ran in ~0.5 secs.

3. Implemented a program for doing 2-dimensional median filtering with a rectangular window. Baseline CPU model had a runtime of ~4secs and the improved version with optimized memory access pattern ran in ~0.7 secs.

4. Developed a Convolutional Neural Network to classify images in C++. Optimized baseline model running on CPU had a runtime of ~4secs and the improved GPU version ran in ~0.8secs.

5. Implemented a parallel sorting algorithm with used the basic approach used in Merge-Sort and Quick-Sort.
